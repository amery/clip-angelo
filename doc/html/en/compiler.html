Chapter 6. Compiler

CLIP compiler (clip) is used to:

  • compile one or more source files (.prg) to an object file form (.o);

  • compile one or more source files (.prg) to a P-code (pseudocode) module form (.po);

  • link one or more source files (.prg), object files (.o), static libraries (.a), dynamic libraries (.so) or a P-code modules (.po) to an executable file form.

6.1. Usage

clip [<CLIP compiler options>] [<filelist>] [<C compiler options>]

where

  • <CLIP compiler options> is a list of one or more options to control the course of the compilation and/or linking, all of which are described below;

  • <filelist> is a space separated list of source, object, library files to compile or link. All file names must be written with suffices, no assumtions about file type are made;

  • <C compiler options> are passed to the C compiler.
