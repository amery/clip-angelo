5.5. Patching your distribution

On ftp server you can find patch.tgz That file contains the last cumulative patch for clip_dev package. Just unpack it (this will overwrite some file) and remake (by the same way as described in "Selective installation" section).

Other libraries and packages some time updating also. Their installation not different from described in "Selective installation".
