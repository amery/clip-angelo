Class UIHBOX

Name

UIHBOX  --  Class of horizontal placement panel.

Synopsis

 UIHBox([<o>],[<spacing>],[<padding>])	--> UIHBox object

Description

Class UIHBox

Attributes

<className> - Class name. Value should be "UIHBox".
<elem> - Widgets on the panel.

Methods

ADD Append widget to panel from right side.
ADDEND Append widget to panel from left side.
SETALIGNMENT Set alignment of widget on the panel.
SETEQUALSIZE Set equal cells size for any widget on the panel.
SETPADDING Set width of panel border.
SETSPACING Set space between widgets on the panel.
UIHBOX Constructor. Create horizontal placement panel.

Method UIHBOX:ADD()

 add(<o>,[<expand>],[<fill>],[<padding>])

<o>

- Widget object. Widget inserted on the panel.

<expand>

- Boolean. Possibility of expand widget horizontally on window size changed.

<fill>

- Boolean. Ability of fill by widget all free space on panel.

<padding>

- Number. Space between widget and cell border in pixels.

Returns :NIL

Method UIHBOX:ADDEND()

 addEnd(<o>,[<expand>],[<fill>],[<padding>])

<o>

- Widget object. Widget inserted on the panel.

<expand>

- Boolean. Possibility of expand widget horizontally on window size changed.

<fill>

- Boolean. Ability of fill by widget all free space on panel.

<padding>

- Number. Space between widget and cell border in pixels.

Returns :NIL

Method UIHBOX:SETALIGNMENT()

 setAlignment(<align>,<valign>)

<align>

- Number. Horizontal alignment of widget. Possible values:

  • ALIGN_LEFT — aligns with the left edge (default)

  • ALIGN_CENTER — centers horizontally

  • ALIGN_RIGHT — aligns with the right edge

<valign>

- Number. Vertical alignment of widget. Possible values:

  • ALIGN_TOP — aligns with the top (default)

  • ALIGN_MIDDLE — centers vertically

  • ALIGN_BOTTOM — aligns with the bottom

Returns :NIL

Method UIHBOX:SETEQUALSIZE()

 setEqualSize(<isEqual>)

<isEqual>

- Boolean. Flag for equal size of cells on the panel.

Returns :NIL

Method UIHBOX:SETPADDING()

 setPadding(<space>)

<space>

- Number. Width of panel border.

Returns :NIL

Method UIHBOX:SETSPACING()

 setSpacing(<space>)

<space>

- Number. Space between widgets on the panel in pixels.

Returns :NIL

Method UIHBOX:UIHBOX()

 UIHBox([<o>],[<spacing>],[<padding>])	--> UIHBox object

<o>

- Widget object. Widget in which panel is inserted.

<spacing>

- Number. Space between cells on panel in pixels.

<padding>

- Number. Space between widget and cell border on panel in pixels.

Returns :NIL

Example:

 box  := UIHBox()
 box:add( UILabel( "Label" ) )

Platforms

No dependies of platform.
