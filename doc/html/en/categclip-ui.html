Function GETDRIVER() 

 getDriver([<name>],[<params>])	--> <oDriver>

<name>

<params>

Function GETSYSIMAGE() 

 getSysImage(<id>)	--> <sImage>

<id>

- Number. Identifier of predefined images.

  • IMG_EMPTY — empty (transparent) pixmap (16x16)

  • IMG_OK — symbol "i" in blue circle (32x32)

  • IMG_ERROR — symbol "x" in red circle (32x32)

  • IMG_WARNING — yellow triangle with exclamation mark (32x32)

  • IMG_QUESTION — symbol "?" in blue circle (32x32)

Returns :

Content of specified predefined images.

See also : UIIMAGE

Example :

 imgContent := getSysImage( IMG_OK )

Function MAPGET() 

 mapget(<m>,<key>,<defVal>)	--> <val>

<m>

- Associated array. Associated array.

<key>

- String. Key.

<defVal>

- Any type. Value by defult on failed key lookup.

Returns :

Value by specified key or <defVal>.

Example :

Function SPLITGEOM() 

 splitGeom(<str>,[<alen>])	--> <aGeometry>

<str>

- String. String of values delimited by commas.

<alen>

- Number. Required number of elements in array. If necessary array size is corrected.

Returns :

Array of values.

Example :

 a := splitGeom( "22", 2 ) 	// {22,NIL}
