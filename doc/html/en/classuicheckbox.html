Class UICHECKBOX

Name

UICHECKBOX  --  Checkbox class.

Synopsis

 UICheckBox([<value>],<label>)	--> UICheckBox object.

Description

Class UICheckBox.

Attributes

<className> - Class name. Value should be "UICheckBox".

Methods

GETVALUE Get check box state.
SETVALUE Check/Uncheck check box.
UICHECKBOX Constructor.

Method UICHECKBOX:GETVALUE()

 getValue() 	--> <value>

Returns :

Returns value of checkbox.

Method UICHECKBOX:SETVALUE()

 setValue(<value>)

<value>

- Boolean. Set check box state: .T. - check box is checked, .F. - check box is unchecked.

Returns :NIL

Method UICHECKBOX:UICHECKBOX()

 UICheckBox([<value>],<label>)	--> UICheckBox object.

<value>

- Boolean. Set check box state: .T. - check box is checked, .F. - check box is unchecked.

<label>

- String. Check box label. You can define accel key for widget activation. Just place symbol "&" before accel letter. This letter will be underlined and widget will be activated with key Alt-letter.

Returns :NIL

Example:

 checkbox := UICheckBox(.T., "&Use log")

Platforms

No dependies of platform.
