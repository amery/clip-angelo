Class UICOMBOBOX

Name

UICOMBOBOX  --  Class of combo box.

Synopsis

 UIComboBox([<values>],[<defaultItem>])	--> UIComboBox object

Description

Class UIComboBox.

Attributes

<className> - Class name. Value should be "UIComboBox".
<list> - List of values. Type: array of strings.

Methods

GETVALUE Get selected string from combo box.
SETLIST Fill list of values.
SETVALUE Select element by default from list.

Method UICOMBOBOX:GETVALUE()

 getValue()	--> <sValue>

Returns :

Selected string from combo box. Returned value is string.

Method UICOMBOBOX:SETLIST()

 setList(<values>)

<values>

- Array of strings. Array of strings as elements of combo box.

Returns :NIL

Method UICOMBOBOX:SETVALUE()

 setValue(<value>)

<value>

- String|Number. String as element of list or serial number of list element (numbering begins from 1).

Returns :NIL

Example:

 cb1 := UIComboBox({"cat","dog"},"dog")
 cb2 := UIComboBox()
 cb2:setList({"apple", "banana"})
 cb2:setValue(1)

Platforms

No dependies of platform.
