Class UIFONT

Name

UIFONT  --  Font class.

Synopsis

 UIFont(<family>,<style>,<size>,<encoding>)	--> UIFont object

Description

Class UIFont

Attributes

<font> - String of X Window System font description.
<family> - Font family.
<style> - Font style.
<size> - Font size in points.
<encoding> - Codepage of the font.

Methods

PARSEFONT Generate X Window System font description and put it in object field 'font'.
SETPROPERTY Change font property.
UIFONT Constructor.

Method UIFONT:PARSEFONT()

 parseFont([<string>])

<string>

- String. String of X Window System font description.

Returns :NIL

Method UIFONT:SETPROPERTY()

 setProperty(<property>,<value>)

<property>

- String. Property name. Possible values: 'family' (font family), 'style' (font style: normal, bold, italic, bolditalic), 'size' (font size in points) and 'encoding' (codepage of the font).

<value>

- String. Value of property.

Returns :NIL

Method UIFONT:UIFONT()

 UIFont([<family>],[<style>],[<size>],[<encoding>])	--> UIFont object

<family>

- String. Font family or string of X Window System font description like "-monotype-arial-medium-r-normal--12-120-0-0-p-0-iso8859-1".

<style>

- String. Font style. Available values: normal, bold, italic, bolditalic.

<size>

- String. Font size in points.

<encoding>

- String. Codepage of the font.

Returns :NIL

Example:

 // Set font Herlvetica-BoldItalic, 12 pt for widget obj
 wfont := UIFont( "Helvetica", "bolditalic", 12 )
 style:font := wfont:font

Platforms

No dependies of platform.
