Class UIFRAME

Name

UIFRAME  --  Class of frame.

Synopsis

 UIFrame([<caption>],[<type>])	--> UIFrame object

Description

Class UIFrame

Attributes

<className> - Class name. Value should be "UIFrame".

Methods

ADD Set placement widget inside frame.
SETLABEL Set frame label.
SETTYPE Set frame type.
UIFRAME Constructor. Create frame with label.

Method UIFRAME:ADD()

 add(<grid>)

<grid>

- Placement object. Set placement widget inside frame. Possible classes for placement object: UIGrid, UIHBox, UIVBox.

Returns :NIL

Method UIFRAME:SETLABEL()

 setLabel(<label>)

<label>

- String. Label at top of the frame.

Returns :NIL

Method UIFRAME:SETTYPE()

 setType(<type>)

<type>

- Number. Frame type.

  • FRAME_PLAIN — no border

  • FRAME_SUNKEN — all frame is sunken

  • FRAME_RAISED — all frame is raised like a button

  • FRAME_IN — only sunken border (default)

  • FRAME_OUT — only raised border

Returns :NIL

Method UIFRAME:UIFRAME()

 UIFrame([<caption>],[<type>])	--> UIFrame object

<caption>

- String. Label at top of the frame.

<type>

- Number. Frame type.

  • FRAME_PLAIN — no border

  • FRAME_SUNKEN — all frame is sunken

  • FRAME_RAISED — all frame is raised like a button

  • FRAME_IN — only sunken border (default)

  • FRAME_OUT — only raised border

Returns :NIL

Example:

 box := UIVBox()
 frame := UIFrame("Section", FRAME_RAISED)
 frame:setGrid(box)

Platforms

No dependies of platform.
