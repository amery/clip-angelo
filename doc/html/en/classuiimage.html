Class UIIMAGE

Name

UIIMAGE  --  Class of image.

Synopsis

 UIImage(<file>,[<isString>])	--> UIImage object.

Description

Class UIImage

Attributes

<className> - Class name. Value should be "UIImage".

Methods

UIIMAGE Constructor. Create image widget.

Method UIIMAGE:UIIMAGE()

 UIImage(<file>,[<isString>])	--> UIImage object.

<file>

- String or number. File name, identifier of system image or content of image file. Identifiers are described in function getSysImage().

<isString>

- Boolean. Flag to interpret file parameter as image content.

Returns :NIL

Example:

 img  := UIImage("icon.xpm")

Platforms

No dependies of platform.
