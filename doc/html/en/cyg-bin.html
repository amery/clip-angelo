5.7. Clip Cygwin binaries

For introduction to Clip you can just install bineries win32 version of Clip. Needed file named clip_win32_bin.rar and it be precompiled equivalent of current release (clip-prg-<version>.tgz)

Just go by the same way as Cygwin installation and set some variables through c:\autoexec.bat:

set CLIPROOT=/clip
set LANG=ru_RU.CP1251
#or
set CLIP_LANG=ru_RU.CP1251
If you use truncated CygWin from ftp.itk.ru - add to autoexec.bat:
set PATH= c:\cygwin\bin;c:\cygwin\clip\bin;
