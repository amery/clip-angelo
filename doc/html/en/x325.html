4.3. Environment variables are using for defines, where CLIP-files are placed:

CLIPROOT

root for all CLIP-related files (default== '$HOME/cliproot' in case of local installation)

CLIP_LOCALE_ROOT

root for the locale messages (default == CLIPROOT)

CLIP_MODULE

current locale module (default=='environ'). In a runtime it's value can be accessed through the predefined macros __CLIP_MODULE__
