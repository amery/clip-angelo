Class UIMAINWINDOW

Name

UIMAINWINDOW  --  Class of main application window. It is inherited from class UIWindow. On main application window close application will be exited.

Synopsis

 UIMainWindow([<caption>],[<name>])	--> UIMainWindow object

Description

Class UIMainWindow

Attributes

<className> - Class name. Value should be "UIMainWindow".
<sysMenus> - Associated array of system menus.

Methods

UIMAINWINDOW Constructor. Create main application window.

Method UIMAINWINDOW:UIMAINWINDOW()

 UIMainWindow([<caption>],[<name>])	--> UIMainWindowText object

<caption>

- String. Window title.

<name>

- String. Window name.

Returns :NIL

Example:

 win  := UIMainWindow("Main window")

Platforms

No dependies of platform.
