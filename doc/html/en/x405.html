4.6. Setting of environment variables.

Environment variables could be set by usual OS- or command interpreter-way or list in $CLIPROOT/etc/environment file, and in the language and the terminal dependance files: lang/$LANG, lang/$LC_ALL, lang/$CHARSET, term/$TERM.

Some variables have duplicates in the command line switches. Settings on the command line override like settings in the environment variable.
